# automssqlbackup
Backup MSSQL Server databases by executing a Powershell script as a Scheduled Task under Windows. 

Features: 
* Daily rotation
* Weekly rotation
* Monthly backups. 
* Optional zipping. 
* Logging. 
* Mail notification.
